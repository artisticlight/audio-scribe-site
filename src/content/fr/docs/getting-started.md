# Prise en Main

Bienvenue sur Audio Scribe. Ce guide vous aidera à tout configurer rapidement.

## Installation

Téléchargez Audio Scribe depuis l'App Store sur votre Mac, iPad ou iPhone.

## Premier Enregistrement

1. Ouvrez Audio Scribe
2. Appuyez sur le bouton d'enregistrement
3. Parlez naturellement — l'application s'occupe du reste
4. Une fois terminé, appuyez sur arrêter pour voir votre transcription

## Choisir un Modèle

Audio Scribe prend en charge les modèles de transcription sur l'appareil et dans le cloud :

- **Sur l'appareil** : Fonctionne hors ligne, entièrement privé. Idéal pour un usage quotidien.
- **Modèles cloud** : Meilleure précision pour le vocabulaire spécialisé. Nécessite une connexion internet.

Vous pouvez changer de modèle à tout moment dans les Réglages.
